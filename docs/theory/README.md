## 📊 Quick Reference

### Key Equations

**Overall Reaction:**
```
2H₂O → 2H₂ + O₂
ΔH° = +285.8 kJ/mol
```

**Theoretical Voltage:**
```
E° = 1.23 V at 25°C
```

**Practical Operating Range:**
- Voltage: 1.8 - 2.2 V per cell
- Current Density: 0.6 - 2.0 A/cm²
- Temperature: 50 - 80°C
- Efficiency: 65 - 80% (LHV)
